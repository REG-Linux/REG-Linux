# fheroes2 (Free Heroes of Might and Magic 2)

Reimplementation of the classic strategy game.

## Build notes
- **Version:** 1.1.12 release.
- **Config:** selects `SDL2`, `SDL2_image`, and `SDL2_mixer`.
- **Build system:** CMake release build with static libs enabled and shared libs disabled; installs the binary using the standard package helper.
