# libretro-xrick

Rick Dangerous core derived from the `xrick-libretro` repository.

## Build notes
- **Version:** commit `476a9a637a6d2afa3f9f6b202bb98b522a4e95d1` (Jan 2025).
- **Config:** requires a C++ toolchain.
- **Build system:** runs the custom `Makefile.libretro` with `platform="unix"` and installs `xrick_libretro.so` into `/usr/lib/libretro`.
