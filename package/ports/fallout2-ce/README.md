# Fallout2-CE

Open-source Fallout 2 engine reimplementation.

## Build notes
- **Version:** v1.3.0 release.
- **Config:** selects `SDL2`.
- **Build system:** CMake release build that produces shared and static libraries, installs the binary under `/usr/bin/fallout2-ce`, and copies the `fallout2-ce.keys` file into `/usr/share/evmapy`.
