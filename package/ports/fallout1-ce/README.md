# Fallout1-CE

Open-source Fallout 1 engine reimplementation.

## Build notes
- **Version:** v1.1.0 release from `alexbatalov/fallout1-ce`.
- **Config:** only selects `SDL2`.
- **Build system:** CMake release build that installs the single binary under `/usr/bin/fallout1-ce` and copies `fallout1-ce.keys` into `/usr/share/evmapy`.
