# Taradino (Rise of the Triad)

SDL2 port of Rise of the Triad (`taradino`).

## Build notes
- **Version:** 20251031 release built from the `taradino` git repo.
- **Config:** selects SDL2 and SDL2_mixer.
- **Build system:** CMake release build with static libs; copies the `rott.keys` evmapy file into `/usr/share/evmapy`.
